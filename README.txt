Keep this program in its own folder.

Make sure this folder contains the following items:
	1. station_location.csv
	2. A folder named "kiosk_data" (omit quotes)
	3. A folder named "month_histograms" (omit quotes)
	4: A folder named "other_graphs" (omit quotes)
    
The "month_histograms" and "other_graphs" folders can remain empty.
The "kiosk_data" has to have .csv files from the bikeshare data to work.

Insert future data into the kiosk_data folder.
Ensure all this data is in .csv format.
